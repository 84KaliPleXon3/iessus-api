Nessus API CLI.

This Python script will interact with a Nessus server using it's robust API. So far, only the common commands have been integrated into this script, however, more will be added over time. To use the script, simply run:

$ python nessus.py -h
$ python nessus.py -l -n -p
$ python nessus.py <args>

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
